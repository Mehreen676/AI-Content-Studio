# AI Content Studio

A powerful AI-powered content creation platform built with Next.js, featuring:

- **6 Content Types**: Blog posts, social media, ad copy, emails, product descriptions, SEO content
- **AI Generation**: Powered by advanced AI models for high-quality content
- **Content Editor**: Built-in editor with AI improvement tools
- **15 Templates**: Professional templates for various content needs
- **Content History**: Save, manage, and revisit your generated content

## Features

- AI Blog Post Generator
- AI Social Media Post Creator
- AI Ad Copy Generator
- AI Email Writer
- AI Product Description Writer
- SEO Content Optimizer
- Content improvement tools (rewrite, expand, shorten, grammar fix)
- Professional content templates
- Content history and favorites

## Tech Stack

- Next.js 16
- Prisma + SQLite
- Tailwind CSS + shadcn/ui
- z-ai-web-dev-sdk for AI features
- Zustand for state management
