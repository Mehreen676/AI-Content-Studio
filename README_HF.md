# AI Content Studio

AI-powered content creation platform built with Next.js, Prisma + SQLite, and z-ai-web-dev-sdk.

## Features
- AI Content Generation (Blog, Social Media, Ads, Emails, Product Descriptions, SEO)
- AI Content Improvement
- Content Management Dashboard
- Templates Library
- Content History & Favorites
- User Authentication

## Tech Stack
- **Frontend**: Next.js 16, React 19, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend**: Next.js API Routes, Prisma ORM, SQLite
- **AI**: z-ai-web-dev-sdk
- **State**: Zustand
